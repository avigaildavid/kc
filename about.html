JavaScript Interview Questions: Object-Oriented JavaScript

QUESTION:
True or False: the terms “scope” and “context” refer to the same thing in JavaScript.
ANSWER:
False
EXPLANATION:
Scope pertains to the visibility of variables, and context refers to the object to which a method belongs (which can be changed by using call or apply).
